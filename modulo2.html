Criando uma conta no GitHub

Módulo 02 - Como criar uma conta no GitHub

Começando no github.com

Agora que entendemos um pouco sobre o Git e GitHub, vamos aprender a como iniciar na plataforma e como utilizar ela de maneira prática!

Você pode criar uma conta na plataforma da seguinte forma. Siga o passo a passo.


Como criar uma conta


1. Acesse o site: github.com

2. Clique em Sing up for GitHub:


3. Preencha os campos: Os campos são feitos em etapas, portanto, conclua uma por vez.

Utilize o e-mail da seguinte forma: seu.email@dominio.com
Exemplo: dev.portal@gmail.com


Após inserir o e-mail, você irá para outra página, que pedirá para criar uma senha.

ATENÇÃO

Certifique-se de que: sua senha tenha pelo menos 15 caracteres OU pelo menos 8 caracteres, incluindo um número e uma letra minúscula.

4. Verificação em duas Etapas (V2E): Verifique seu e-mail, onde o código de verificação será enviado, após verificar, sua conta estará criada.

5. Pronto! Sua conta está criada. Agora você terá acesso ao portal do usuário e terá acesso às funcionalidades citadas no Módulo 1 > Principais funcionalidades .
